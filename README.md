## download

```bash
sudo sh -c "$(curl -fsSL https://get.tunnel.paiya.app)"
```

如果发现 github 被墙了可以直接这样试试

```bash
# 例子: HTTPS_PROXY=http://127.0.0.1:8081

sudo HTTPS_PROXY=你的代理链接 sh -c "$(curl -fsSL https://get.tunnel.paiya.app)"
```
